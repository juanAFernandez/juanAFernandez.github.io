Conceptos

Servidor

Servidor +- = ordenador convencional

Aunque puede ser cualquier equipo hacemos referencia a máquinas muy específicas con:
  • Hardware más preciso.
  • Mayor capacidad de procesamiento y almacenamiento.
  • Sustitución en caliente.

Not StandAlone

Granja

Es inusual tener un servidor trabajando sin equipos con quien interactuar.
  • Routers / switches
  • Firewalls
  • Otros servidores
  • Equipos auxiliares

Clasificación

Realmente "no existe"

Cualquier servidor es una ordenador "grande" que ejecuta algún tipo de software, sea:
  • Un servidor web (Apache, Nginx, Node, etc... )
  • Servicios (BD, Video, etc...)
  • Acceso a recursos (RAID, Impresión, Red,
    Telefonía (Asterisk), etc...)
  • Hardware de cualquier tipo, (Arduino, p.e.).

Sistemas de monitorización

Indicadores visuales (HP Standar)

  • System power LED (Alimentación del sistema)

    Activo (verde) - Standby (ambar)

    Ajuste potencia (1Hz verde)

  • Health LED (Estado del sistema)

    Normal (verde)

    Sistema degradado(flash ambar)

    Sistema crítico (1Hz rojo)

    Sistema no operativo (bios level, agoniza), (4Hz rojo)

  • NIC LED (Estado de tarjeta RED)

    Conectado a la red(verde)

    Red activa (1Hz verde)

    Desconectado (apagado)

  • UID LED

    Administración remota o actualización de firmware (1Hz azul)

BMC & IPMI

BMC

Procesador especializado para monitorear el estado físico del equipo de forma paralela a su funcionamiento.

Crea la interfaz entre el software y el hardware

"Inteligencia en la arquitectua IPMI"

IPMI

Interfaz basada en el paso de mensajes

Opera de forma independiente al SO, por eso funciona: antes de que arranque, cuando está apagado y después de un fallo (entre otros escenarios)

Intel 1998

Más de 200 empresas lo usan, Cisco, Dell, HP, NEC, SuperMicro, etc...

BMC es parte de IPMI y suele estar presente en la placa base

El sistema IPMI puede añadirse si no se encuentra disponible
